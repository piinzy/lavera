# لافيرا | Lavera

ثيم سلة (Salla Twilight) بطابع تحريري راقٍ (Editorial) للأزياء والموضة والمنتجات الفاخرة — مبني فوق ثيم [Raed](https://github.com/SallaApp/theme-raed) الرسمي مفتوح المصدر من سلة (MIT)، ومُعاد بناؤه بصريًا بالكامل بروح ثيمات المجلات العالمية (Impression / Mimi).

## الهوية البصرية

| العنصر | القيمة |
|---|---|
| خط العناوين | Cormorant Garamond (قابل للتغيير من لوحة التاجر) |
| خط النصوص | Markazi Text — سيريف عربي أنيق (قابل للتغيير) |
| الأرضية | أبيض نقي، لوحات بإطارات رفيعة `#eeeeee` بدل الظلال |
| لون التخفيضات | `#e84e4e` (قابل للتغيير من لوحة التاجر) |
| الزوايا | حادة بالكامل — طابع مجلات الأزياء |
| الاتجاه | RTL أولًا، مع دعم كامل لـ LTR |

## مكونات الصفحة الرئيسية (10)

يبني التاجر صفحته الرئيسية بالسحب والإفلات من لوحة التحكم:

1. **سلايدر محسّن** — صورة مخصصة للجوال لكل شريحة، زر CTA، موضع محتوى قابل للاختيار، 3 ارتفاعات
2. **شريط ترويجي متحرك (Marquee)** — نصوص متحركة باستمرار، يتوقف عند المرور، RTL تلقائي، يحترم تقليل الحركة
3. **تبويبات التصنيفات** — حتى 4 تبويبات بخط سفلي متحرك، كل تبويب مربوط بتصنيف
4. **صورة مع نص (تحريري)** — لقصة العلامة، موضع صورة قابل للتبديل
5. **لوك بوك** — إطلالات مع قوائم قطع وأسعار
6. **روابط سريعة**، **سلايدر منتجات**، **بنرات مربعة**، **ماركات**، **آراء عملاء مخصصة**

بالإضافة لمكونات سلة القياسية (منتجات مميزة، يوتيوب، بارالاكس...).

## إعدادات "مظهر لافيرا"

من لوحة التاجر مباشرة:
- لون سعر التخفيض
- أحرف كبيرة متباعدة في القائمة (للمتاجر الإنجليزية)
- تفعيل/إيقاف تكبير صور المنتجات عند المرور
- التباعد الرأسي بين الأقسام (متقارب / مريح / واسع)
- خطا Google Fonts للعناوين والنصوص (اكتب الاسم كما هو)

## التغطية

هوية موحدة عبر: الرئيسية، صفحة المنتج (معرض حاد + أسعار سيريف + شريط إضافة ثابت للجوال)، التصنيفات، السلة، آراء العملاء (أسلوب اقتباسات المجلات)، صفحات حساب العميل، المدونة، والصفحات التعريفية.

## التطوير محليًا

```bash
pnpm install
pnpm run watch      # تطوير مع مراقبة التغييرات
pnpm run production # بناء نهائي للنشر
```

المعاينة الحية على متجر تجريبي عبر [Salla CLI](https://docs.salla.dev/421879m0):

```bash
npm i -g @salla.sa/cli
salla login          # بحساب شركاء سلة
salla theme preview
```

## بنية المشروع

```
twilight.json           توصيف الثيم + إعدادات لوحة التاجر + تعريف المكونات
src/views/
  layouts/              master.twig (الغلاف) + customer.twig
  pages/                index, product/, cart, blog/, customer/ ...
  components/
    header/ footer/     الهيدر والفوتر التحريري
    home/               مكونات الصفحة الرئيسية العشرة
src/assets/styles/      SCSS بمنهجية ITCSS + Tailwind
src/assets/js/          سكربتات الصفحات + بطاقة المنتج (Custom Element)
public/                 المخرجات المبنية (webpack)
```

## ملاحظات تقنية

- كل تعديلات المظهر فوق مكونات سلة الرسمية (`salla-*` web components) — لم يُمس أي منطق برمجي للسلة/الدفع/المخزون
- التخصيصات معزولة بكلاسات `lavera-*` أو ملفات `-editorial.scss` مستقلة — سهلة التتبع والتوسعة
- أي إعداد جديد يتبع نفس النمط: حقل في `twilight.json` ← body class أو متغير CSS في `master.twig` ← تنسيق SCSS
