# Lavera

ثيم سلة (Salla Twilight) بطابع تحريري (Editorial) للأزياء والموضة — مبني فوق ثيم [Raed](https://github.com/SallaApp/theme-raed) الرسمي مفتوح المصدر من سلة (MIT)، مُعاد تجليده بصريًا ليطابق روح ثيم Shopify **Impression / بريست Mimi**.

## الهوية البصرية

- **الخطوط:** Cormorant Garamond (عناوين) + Markazi Text (نصوص عربية أنيقة) عبر Google Fonts
- **الألوان:** أبيض/أسود/رمادي فاتح `#f5f5f5`، أحمر التخفيضات `#e84e4e`
- **الزوايا:** حادة بالكامل (لا انحناء) — طابع مجلات الأزياء الراقية
- **RTL:** مدعوم بالكامل (العربية أولًا)

## التشغيل محليًا

```bash
pnpm install
pnpm run watch      # تطوير مع مراقبة التغييرات
pnpm run production # بناء نهائي للنشر
```

ثم اربط الثيم بمتجر سلة عبر [Salla CLI](https://docs.salla.dev/421879m0) (`salla theme:serve`) لمعاينته حيًا على متجرك.
